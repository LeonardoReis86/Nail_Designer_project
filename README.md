## ☕ Sobre o projeto
Página web que destacará o trabalho e a prestação de serviços de uma nail designer, oferecendo um layout responsivo e acessível para uma boa experiência de usuário em dispositivos móveis, tablets e desktops. Com espaço para apresentação do studio, seus conceitos e diferenciais, proporcionando uma visão interessante do ambiente, bem como seus valores. Neste projeto teremos cards de apresentação de serviços, exibindo opções de serviços como aplicação de alongamento, manutenção, esmaltação em gel, entre outros.
